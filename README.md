# Automated Handwritten Str8ts Puzzle Recognition

This repository contains the final report for my project **"Automated Handwritten Str8ts Puzzle Recognition"**, carried out from September 2025 to December 2025 during my studies at the Hong Kong University of Science and Technology. The project focused on detecting and classifying handwritten Str8ts puzzles using deep learning.

## Project description

- Grid detection (Mask R‑CNN, IoU 0.97), perspective correction, and 81‑cell extraction from a self‑created dataset.
- CNN classifier + MLP for black‑cell / clue / digit classification, handling real‑world distortions and handwriting variation.
- Tested on 25 unseen images; optimized via SGD and data augmentation (grid accuracy >90%).
